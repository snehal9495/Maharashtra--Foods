# Maharashtra Foods Premium Website v2.0

GitHub Pages ready static website.

## Publish
1. Create/open your GitHub repository.
2. Upload all files and folders from this project (keep index.html in the root).
3. Commit changes.
4. Go to Settings > Pages.
5. Select Deploy from a branch, `main`, `/ (root)`, then Save.
